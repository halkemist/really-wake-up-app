# really-wake-up-app

## Install packages (first installation / or package updates)
```bash
npm install
```

## Run the app (web)
```bash
ionic serve
```
## Run the app (android)

### Build app
```bash
npm run build
```

### Sync to capacitor files
```bash
npx cap sync
```

### Run android app
```bash
npx cap run android
```

### Run android IDE (studio)
```bash
npx cap open android
```
